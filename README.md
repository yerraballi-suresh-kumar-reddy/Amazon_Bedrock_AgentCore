# Amazon Bedrock AgentCore 

AgentCore = Everything needed to make an AI agent production-ready:

- Runtime: runs your agent
- Built-in Tools: adds powerful capabilities
- Gateways: connects to external APIs
- Memory: lets your agent remember
- Identity: secures your agent
- Observability: lets you monitor everything

It works with any model and any framework.
